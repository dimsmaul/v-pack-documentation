# CLI

## CLI Commands

V-Pack provides a CLI tool for initialization and configuration.

## Init

Initialize V-Pack configuration in your project.

```bash
npx v-pack init
```
Creates ```tailwind.config.js``` in your project root with default theme colors and settings.

## Option

Force Overwrite

```bash
npx v-pack init --force
# or
npx v-pack init -f
```

Overwrites existing ```tailwind.config.js``` file.
